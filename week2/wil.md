# 마크다운 언어 공부 내용 정리

간결하게 텍스트르 구조화하고 스타일을 적용할 수 있게 해줌

2004년에 John Gruber가 개발, 이후 GitHub, Notion, Velog 등 많은 플랫폼에서 채택되어 널리 사용됨.

---

## 문법 정리

### 1. 제목 (Headers)

`#`의 개수로 수준을 조절 (1개부터 6개까지 가능)

- `# 제목 1`
- `## 제목 2`
- `### 제목 3`

### 2. 텍스트 강조 (Emphasis)

- **굵게**: `*텍스트**` 또는 `__텍스트__`
- *기울임*: `텍스트*` 또는 `_텍스트_`
- ~~취소선~~: `~~텍스트~~`

### 3. 목록 (Lists)

- **순서가 없는 목록**: `-`, `*`, `+` 중 하나를 사용
    - `사과`
    - `바나나`
- **순서가 있는 목록**: 숫자를 사용
    1. `1. 첫 번째`
    2. `2. 두 번째`

### 4. 링크와 이미지 (Links & Images)

- **링크**: `[표시할 이름](URL)`
    - 예: `[구글](https://www.google.com)`
- **이미지**: `![대체 텍스트](이미지 주소)`
    - 예: `![로고](https://example.com/logo.png)`

### 5. 코드 (Code)

- **인라인 코드**: 텍스트 중간에 코드를 넣을 때 백틱( ` , option + ~)으로 감쌉니다.
    - 예: ``git clone`` 명령어를 입력하세요.
- **코드 블록**: 여러 줄의 코드를 작성할 때 백틱 3개( ``` )를 위아래로 감쌉니다. 첫 줄에 언어 이름을 적으면 문법 강조(Highlighting)가 됩니다.

```java
print("Hello World!")
```

### 6. 인용문과 구분선 (Blockquotes & Horizontal Rules)

- **인용문**: `>` 기호를 사용합니다.
    - `> 이 내용은 인용문입니다.`
- **구분선**:  또는  또는 `_`를 3개 이상 연속으로 입력합니다.
    - `--`

### 7. 표 (Tables)

수직선(`|`)과 하이픈(`-`)을 조합하여 만듭니다.

`| 항목 | 내용 | 비고 |
| :--- | :---: | ---: |
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
| 데이터 1 | 데이터 2 | 데이터 3 |`
